# The Hardest Job Hunter

C++ 콘솔 환경에서 만든 장애물 피하기 게임입니다.

2000년대에 인기를 끌었던 고난도 플래시 게임인 **The World's Hardest Game**을 참고하여 제작했으며, 취업 준비생이 여러 장애물을 피하고 최종 합격까지 도달하는 내용을 담았습니다.

---

## 프로젝트 소개

플레이어가 맵 안에 있는 장애물을 피하면서 목적지까지 이동하는 게임입니다.

장애물에 닿으면 사망 횟수가 증가하고 처음 위치로 돌아가며, 각 스테이지의 도착 지점에 도달하면 다음 스테이지로 이동합니다.

게임 화면 위쪽에는 현재 시간과 사망 횟수가 표시되도록 만들었습니다.

---

## 프로젝트 주제

기존의 **The World's Hardest Game**을 참고하여 C++ 콘솔 게임으로 다시 제작했습니다.

원작의 장애물 피하기 방식을 사용하면서, 취업 준비생이 어려운 과정을 통과해 최종 합격하는 내용으로 바꾸었습니다.

## 시작 화면

게임 시작 전 메뉴에서 원하는 항목을 선택할 수 있도록 구성했습니다.

## 게임 화면

플레이어는 맵 안에서 움직이는 장애물을 피하며 도착 지점까지 이동해야 합니다.

화면 위쪽에는 게임 진행에 필요한 정보를 표시했습니다.

- 현재 스테이지
- 플레이 시간
- 현재 사망 횟수
- 플레이어 위치
- 남은 기회

장애물에 부딪히면 사망 횟수가 증가하고 플레이어가 시작 위치로 돌아갑니다.

## 스테이지 2 구현

저는 프로젝트에서 **스테이지 2 맵 제작과 장애물 배치**를 담당했습니다.

스테이지 2는 긴 복도와 여러 층으로 나누어진 길을 따라 이동하도록 구성했습니다.

각 장애물의 시작 위치와 이동 방향을 직접 정했으며, 장애물이 벽과 겹치지 않고 맵 안에서 자연스럽게 움직이도록 좌표를 계속 수정했습니다.

## 장애물 배치 코드

장애물은 행과 열 좌표를 이용해 맵에 배치했습니다.

마지막 값인 `1`과 `-1`을 사용하여 장애물이 오른쪽 또는 왼쪽으로 이동하도록 설정했습니다.

```cpp
Obs init[] = {
    {3, 42, 0, 1},
    {3, 46, 0, -1},
    {7, 45, 0, 1},
    {7, 60, 0, -1},

    // 중간 긴 복도
    {20, 20, 0, 1},
    {21, 95, 0, -1},
    {22, 30, 0, 1},
    {23, 85, 0, -1},

    // 아래쪽 맵 구역
    {28, 30, 0, 1},
    {29, 80, 0, -1},
    {30, 35, 0, 1},
    {31, 75, 0, -1}
};
